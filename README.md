# CV Application

A React application for building and exporting CVs/Resumes.

preview: https://cvgenerator-omega.vercel.app/

## Features

- Multi-step form wizard (4 pages)
- General details (name, email, phone)
- Education details with add/remove multiple entries
- Experience details with add/remove multiple entries
- Review page with PDF export
- Form validation
- Page indicator

## Tech Stack

- React 19
- Vite
- Tailwind CSS
-

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
